# NvPak: Clean & Modern Neovim (2026)

[![Neovim](https://img.shields.io/badge/Neovim-0.10.0-green.svg)](https://neovim.io)

## Highlights
- **Pak (Clean):** Minimalist core with high performance.
- **Modern:** Built with `blink.cmp` and `snacks.nvim`.
- **Persian Ready:** Built-in RTL support for Persian/Arabic users.

## Requirements
- Neovim >= 0.10.0
- Nerd Font (Recommended: Vazirmatn RD Nerd Font)

## RTL Support
Toggle Persian RTL mode with `<leader>rtl`. Optimized for Konsole/Plasma environments.
